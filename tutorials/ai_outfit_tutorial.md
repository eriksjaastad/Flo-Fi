# **How to Design AI Character Outfits**

*Based on tutorial by @daisystudios.ai*

## **Overview**

This workflow uses three AI platforms together to create consistent, stylized character outfits. Each platform serves a specific role:

* **Midjourney** — for style and aesthetic quality  
* **Nano Banana** — for continuity and outfit accuracy  
* **Sea Dream** — for continuity with less censorship

## **Key Rules to Follow Throughout**

* Always use the same original image under Omni Reference — never use previously generated images  
* Omni Reference is slightly off each time, so consistency of the source image is critical  
* You can stop at any step if the result is good enough — the later steps are for refinement

## **Method 1: Simple Single-Color Outfit**

Best for: solid color outfits like a red rubber jumpsuit. Straightforward and hard to mess up.

1. Place your character image in **Omni Reference** in Midjourney  
2. Place that same character image as your **Image Prompt**  
3. Find a reference image of the outfit you want from the internet  
4. Place the outfit reference image in **Style Reference**  
5. Generate — you should get a solid result

## **Method 2: Complex Outfit (e.g. Catwoman)**

Best for: detailed costumes with multiple elements. Uses all three platforms for best results.

### **Phase 1 — Get a Base Image in Midjourney**

6. Place your character in **Omni Reference**  
7. For the outfit, use one of these three options (all give decent results):  
* Add the costume image to Style Reference or Image Prompt  
* Describe the costume in text  
8. Generate until you get a half-decent base — you just need the outfit shape, not perfection

### **Phase 2 — Refine in Nano Banana or Sea Dream**

9. Take the base image from Midjourney into **Nano Banana** or **Sea Dream**  
10. Add the reference image of the costume you want your character wearing  
11. Generate — results will be accurate but may look ordinary/generic in style

### **Phase 3 — Polish Back in Midjourney**

12. Download the Nano Banana output  
13. Go back to Midjourney and use your **original character image** as Image Prompt  
14. Add the Nano Banana output as an additional image reference  
15. Generate — this brings back Midjourney's style while keeping the outfit accuracy

### **Phase 4 (Optional) — Dial In Details**

16. Take the image back to Nano Banana  
17. Type in small detail requests like zippers, stitching, accessories  
18. Bring that back to Midjourney one final time for the style finish

## **Method 3: Text Description Approach (e.g. Wonder Woman / Supergirl)**

Best for: when you have a clear vision and want to describe the outfit rather than reference an image.

19. Start in Nano Banana or Sea Dream with your Omni Reference character image  
20. Write a detailed text description of everything you want (e.g. skirt not pants, arm band, headband)  
21. Generate and note anything missing  
22. Keep adding details in text until satisfied  
23. Bring the result back to Midjourney using your original Omni Reference image  
24. Add the Nano Banana image as image reference and generate for final stylized result

## **Platform Quick Reference**

* **Midjourney:** Style, aesthetics, visual quality  
* **Nano Banana:** Outfit accuracy and continuity, fine detail control  
* **Sea Dream:** Similar to Nano Banana, less censorship
