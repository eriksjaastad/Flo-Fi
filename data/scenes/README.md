# Scenes

Scene backlog for Flo content. Each scene lives as a single markdown file and moves through the pipeline:

```
backlog/       # storyboard written, not yet in production
in_progress/   # actively grinding stills/videos/voice/lip-sync
done/          # shipped
```

**Move files (don't copy)** as scenes advance. One file = one scene, one location at a time.

## Scene doc structure

Every scene file should contain:
- **Logline** — one sentence
- **Setting** — where/when/lighting
- **Beats** — story beats in order
- **Shots / keyframes** — what the camera sees, shot by shot
- **Dialogue** — voice lines with delivery notes
- **Sound design** — diegetic sounds, silence, cues
- **Open questions** — unresolved design decisions blocking production

## Production rule

Do NOT start grinding Midjourney clips for a scene until:
1. Every shot has a keyframe description
2. All recurring character/prop designs referenced are locked (e.g., drone geometry)
3. Dialogue is written (no improvising on the mic — see `feedback_voice_recording.md`)
